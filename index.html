What is Project Xenon™?

Project Xenon™ is a technology preview of a decentralized systems development framework.

  • Lightweight, Decentralized Services - Quickly build orchestration in the form of cooperating REST services using a shared data fabric across many nodes;
  • Built-in Query/Index/Filter - Query and filter across decentralized document-oriented services;
  • Decentralized Persistence, Consensus, and Replication - create stateful distributed services with built-in concurrency, tunable replication, and persistence;
  • Dynamic Discovery, Declarative API and RESTful Services - orchestrate between multiple services, build simple document-centric REST APIs, and easy-to-use clients;

Getting Project Xenon™

Project Xenon™ source code is available on the VMware Project Xenon™ GitHub source repository. You can build Project Xenon™ by cloning the repo and running mvn install.

Support

Project Xenon™ is released as open source software and comes with no commercial support.

But since we want to ensure success and recognize that Project Xenon™ consumers might fall into a range of roles - from developers that are steeped in the conventions of open-source to customers that are more accustomed to VMware commercial offerings, we offer several methods of engaging with the Project Xenon™ team and community.

For our developer community, feel free to join our Google groups at vmware-photon-os-dev

For more general user questions, visit the Project Xenon™ user forum in our Project Xenon™ VMware Community.

Contributing

The Project Xenon project team welcomes contributions from the community. If you wish to contribute code, we require that you first sign our Contributor License Agreement and return a copy to osscontributions@vmware.com before we can merge your contribution.
